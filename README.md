# Oryzenx - Premium Domain Marketplace

Enterprise-grade domain marketplace platform with advanced blog system, payment processing, and comprehensive admin panel.

## Features

### Domain System
- Complete CRUD operations
- Domain pricing and bidding
- Rating and quality badges
- Featured domains
- Real-time offer system

### Blog Platform
- Dynamic CMS with rich text editor
- Multi-language support (English, Bengali)
- SEO optimization
- Comments and reactions
- Category and tag management

### Requirements

- PHP 8.0 or higher
- MySQL 5.7 or higher
- Apache with mod_rewrite

## Installation

1. Extract files to web root
2. Visit install.php
3. Follow the wizard
4. Configure settings
